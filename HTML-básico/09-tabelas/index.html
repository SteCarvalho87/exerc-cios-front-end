Tabelas

Hoje vamos criar tabelas.

  1. Primeiro usamos a tag "table" para indicar que será feito uma tabela
  2. Depois criamos as linhas com a tag "tr" aninhada na table
  3. Aninhado na primeira linha fazemos as tags "th" que indicam o cabeçalho de cada coluna
  4. aninhadas nas próximas linhas fazemos as tag "td" que são os dados de parte da tabela

Olhando a tabela acima ela não fica muito bonitinha mas é uma tabela. Pra deixar ela arrumadinha são feitos comandos no CSS

Existe uma forma de usar o emmet para adiantar a criação de uma tabela. A tabela acima por exemplo poderia ser criada com o seguinte código "table>tr*3>td*5". Traduzindo, crie uma tabela com 3 linhas e 5 colunas dentro de cada linha
