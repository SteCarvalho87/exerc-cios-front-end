CSS Interno

Vamos estilizar a página

O CSS iterno é digitado na head numa estilização CSS o padrão é o nome da tag mais chaves encerrando com ponto e vírgula: tag { xxxx; }

Vantagens e desvantagens

A vantagem de ter um ponto único de mudança é que na manutenção da página basta modificar nesse ponto e o estilo é aplicado à pagina toda A desvantagem é que se o site possuir mais de uma página você continua conseguindo modificar somente a página onde o estilo esta aplicado
